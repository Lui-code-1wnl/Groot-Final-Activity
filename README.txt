# GROOT-FINAL-ACTIVITY

## Overview

This repository contains the source code for the websites. 
The project is divided into two parts: one written in PHP and the other 2 in Node.js.

## PHP Website
### Prerequisites

- PHP installed on your machine ([Download PHP](https://www.php.net/downloads.php))

### How to Run

1. Open a terminal/command prompt.
2. Navigate to the `php-website` directory.
3. Run the following command:
   ```bash
   php -S localhost:8000
Open your web browser and go to localhost/Groot-Final-Activity to view the PHP website.

## PHP Website
Node.js Website

Node.js installed on your machine (Download Node.js)
How to Run
